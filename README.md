### Package Installation - Add package dependencies in package.json

#### *Note*: <h5>Make sure you have to choose compatible version</h5>
```
"@angular-builders/custom-webpack": "8.1.0",
"@types/node": "8.9.5",
"@types/webpack": "^5.28.5",
"dotenv": "8.1.0"
```
